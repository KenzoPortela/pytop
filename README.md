# PyTop - htop in Python

A dynamic, htop-like system monitor for Windows with real-time CPU, GPU, memory monitoring and process management, made in Python

## Features

- Real-time CPU core monitoring with frequency display
- Memory and GPU/VRAM usage tracking
- Process list sorted by CPU usage
- Kill processes with keyboard shortcuts
- Responsive layout that adapts to terminal size

## Installation

See the avaliable wheel (`.whl`) and archive (`tar.gz`) files in the release section

PyPi validation pending, to allow install via `pip install` command, so stay tuned!

## Usage

Simply run:
```bash
pytop
```

## Controls

- `q` - Quit
- `k` - Kill selected process
- `↑/↓` - Navigate process list

## Requirements

- Python 3.7+
- Windows
